1. Для запуска программы нужно установить необходимые библиотеки командой: go mod download
2. В файле .env написать настройки подключения к базе данных. 
3. Запуск программы без докера командой: go run cmd/main.go
4. Просмотр swagger документации http://localhost:8081/swagger/index.html
5. Запуск в докере командой: docker-compose up --build
